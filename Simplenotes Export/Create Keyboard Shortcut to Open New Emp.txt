Create Keyboard Shortcut to Open New Empty Doc in Pluma ✏️

--add tags--

You can create a custom keyboard shortcut that runs a shell command to create a new empty file and open it in Pluma.

**Follow these steps:**

**1. Decide on the Command**

You want a command that creates a new empty file (for example, in your `~/Documents` folder) and opens it in Pluma. Here’s a suitable command:

```sh
pluma "$(mktemp ~/Documents/Untitled_XXXXXX.txt)"
```

- `mktemp ~/Documents/Untitled_XXXXXX.txt` creates a unique named empty file in your Doc folder.
- `pluma ...` opens that file in Pluma.

**2. Add the Shortcut in Your Desktop Environment**

For Xfce:

- Go to **Settings > Keyboard > Application Shortcuts**.
- Click **Add**, paste the command, and set your key.

**3. Test the Shortcut**

Press your chosen key combination. Pluma should open a new, uniquely named empty document in your Documents folder.

---

**Note:**  
- The shortcut setup interface may vary slightly by desktop environment, but the process is similar[3][4][5].
- The command uses `mktemp` to avoid overwriting existing files and to ensure each new document is unique.

---

If you want the file to always have the same name (and be overwritten), use:

```sh
touch ~/Documents/Untitled.txt && pluma ~/Documents/Untitled.txt
```

But using `mktemp` is safer for multiple new documents.
