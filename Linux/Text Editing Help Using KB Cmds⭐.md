### Ctrl + Shift + End
- **Function:** selects all text from current cursor position to end of document/text field.
- **Use:** highlight everything below cursor - similar action as above.

### Ctrl + Shift + Home
- **Function:** selects txt from cursor position to start of doc/ext field.
- **Use:** quickly highlight everything above cursor to copy, cut, format.
