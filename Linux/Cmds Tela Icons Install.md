Tela Icons Install Cmds
---

`./install.sh -c -d /usr/share/icons standard pink nord`

Explanation:

***-c***: Installs the circle folder version.
***-d /usr/share/icons***: Sets the destination directory.
***pink***: Specifies color variants to install, single or multi 
e.g. ***pink standard nord***:   

### Installation

Usage:  `./install.sh`  **[options...]** **[colour variants...]**

| Options    | Info                                                            |
|:------------:|:------------------------------------------------------------------------------|
| `-a`        | Install all color versions                                                    |
| `-d`        | Specify theme destination directory (**Default:** _$HOME/.local/share/icons_) |
| `-n`        | Specify theme name (**Default:** _Tela-circle_)                               |
| `-c`        | Install circle folder versions                                                |
| `-h`        | Show this help                                                                |

&nbsp;

| colour variants   | Info                           |
|:------------------|:--------------------------------------|
| `standard`        | Standard folder colour         |
| `black`           | Black folder colour            |
| `blue`            | Blue folder colour             |
| `brown`           | Brown folder colour            |
| `green`           | Green folder colour            |
| `grey`            | Grey folder colour             |
| `orange`          | Orange folder colour           |
| `pink`            | Pink folder colour             |
| `purple`          | Purple folder colour           |
| `red`             | Red folder colour              |
| `yellow`          | Yellow folder colour           |
| `manjaro`         | Manjaro default folder colour  |
| `ubuntu`          | Ubuntu default folder colour   |
| `dracula`         | Dracula default folder colour  |
| `nord`            | Nord default folder colour     |
