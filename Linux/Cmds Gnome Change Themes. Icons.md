Change Themes icons Cursors in Gnome

Icons:
gsettings set org.gnome.desktop.interface icon-theme "Icon-Name-Here"

Themes:
gsettings set org.gnome.desktop.interface gtk-theme 'Theme-Name-here'

Cursors:
gsettings set org.gnome.desktop.interface cursor-theme "Cursor-Name-here"

**Note: Icon names are taken from the folder they are in**
